## Install
download source code: http://www.apuebook.com/code3e.html


## Enviroment
    $ tar -zxvf *.tar.gz
    $ cd apue.3e/
    $ sudo apt-get install libbsd-dev  
    $ sudo cp ./include/apue.h /usr/include/  
    $ sudo cp ./lib/libapue.a /usr/local/lib/
    
## Compile
    $ gcc 1-3.c -o 1-3 -lapue -Werror
